Les jets s'afficheraient dans le chat avec un template (mise en forme) s'inscivant dans une tache de tasse à café (je pensais utiliser cette image : https://raw.githubusercontent.com/NathaTerrien/roll20-wip/master/Macchiato%20Monsters/mm_bg_tmplt.png )

Le template de résultat affiche des infos minimales : nom du jet (la stat, l'arme, l'équipement etc...), le résultat du jet et le nom du personnage (cf. maquette ci-dessous).

Aucun jet ne demande de bonus/malus/choix : on clique et le résultat apparait dans le chat.

Les jets de stats et de dégâts sont jetés sytématiquement deux fois (avantage et désavantage) et les deux résultats affichés en même dans le chat (cf. fiche Terra-X).
Pour les jets de stats, sous chaque jet, un indicateur de succès ou réussite.
On utilise le résultat de gauche pour un jet simple.
Cf maquette : https://raw.githubusercontent.com/NathaTerrien/roll20-wip/master/Macchiato%20Monsters/tmplt_model.jpg

Les jets de dR afficherait un truc si <= à 3 : si on fait les choix de la liste déroulante de dé (cf. ci-dessus), on peut afficher dans le template le dé "réduit" (ex : on fait un jet de dR6, résultat 2, on peut afficher un message du genre "Risk Dice is reduced to d4"). Si on fait le choix de zone de saisie, et en cas de résultat <= 3, on peut juste afficher un message sans donner le dé résultant (on ne sait pas que ce qu'on a jeté en fait).
